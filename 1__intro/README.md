# Curso de React — Sessão 01

## Introdução

Nesta primeira sessão foram apresentados os conceitos iniciais para começar a trabalhar com React, com foco na preparação do ambiente de desenvolvimento e na criação dos primeiros projetos.

## Instalação do Node.js

O Node.js é necessário para executar JavaScript fora do navegador e gerenciar dependências.

## NPM 

O Npm é o gerenciador de pacotes do Node.js, utilizado para instalar bibliotecas e ferramentas.


## Criação de projeto React 

Aqui utilizei o comando default de criação de projetos React: $ npx create-create-app <nome>
Logo em seguida iniciei a aplicação com: npm start 

## Criação de projeto React com Vite

Aqui utilizei o comando de criação: npm create vite@latest
Logo em seguida iniciei a aplicação selecionando as opções no terminal.

## Instalação de Extensão 
Para auxiliar na produtivodade instalei a extensão: ES7 + React/Redux/React-Native snippets
